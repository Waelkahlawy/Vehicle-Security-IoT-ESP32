An embedded IoT system based on **ESP32 WROOM  for monitoring vehicle security and ambient lighting conditions.
The system detects motion, proximity, temperature, and light intensity, and provides real-time alerts via **MQTT** and **Blynk Cloud**.

## Features
- Motion and vibration detection (MPU6050)
- Object proximity detection (Ultrasonic Sensor)
- Temperature monitoring (LM35)
- Ambient light sensing (LDR)
- Automatic vehicle lighting control
- Real-time monitoring using Blynk mobile application
- MQTT-based cloud communication
- Optional GSM support (SIM800L)

## Hardware Components
- ESP32 WROOM 
- MPU6050
- Ultrasonic Sensor
- LM35 Temperature Sensor
- LDR
- SIM800L 

## Software & Technologies
- PlatformIO
- MQTT Protocol
- Blynk IoT Platform
- Wi-Fi Communication
- 2G
